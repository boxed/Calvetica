# Calvetica Calendar

The fastest calendar for iPhone and iPad. A beautiful, highly customizable calendar app designed for speed and efficiency.

[![App Store](https://img.shields.io/badge/App_Store-Available-blue)](https://apps.apple.com/us/app/calvetica-calendar/id385862462)

## Features

### Multiple Views
- **Month View** - Rich display showing how events span and overlap across days
- **Week View** - Continuous scrolling week view
- **Detailed Week View** - Week view with week numbers
- **Full Day View** - Focused single-day display
- **Agenda View** - List-based event overview

### Fast Event Creation
- Long-press on any day to quickly add events
- Fast number pad entry for times
- Complex recurring events (e.g., "first Thursday of every month")

### Powerful Alerts
- Custom Calvetica alerts with snooze options
- Louder alert sounds than default iOS alerts
- Multiple default alarms per event
- Smart all-day event reminders (e.g., "remind me at 6pm the day before")

### Customization
- Customizable calendar colors and visibility
- Flexible week start options
- 24-hour format support
- Time zone support
- Customizable event detail screen layout

### Search & Navigation
- Search events by title, notes, and location
- Highlighted search results
- Tappable email addresses and phone numbers in notes

### Sharing
- Share events via iCal email
- SMS event descriptions
- Attendee management
- Map integration for locations

## Development

### Requirements
- Xcode
- CocoaPods

### Setup

```bash
# Install dependencies
pod install

# Open workspace (not the project file)
open calvetica.xcworkspace
```
